# Contributing

Thanks for helping improve OnDevice LLM Toolkit! Contributions are welcome.

## Quick start

1. Fork the repo and create a feature branch.
2. Create a virtual environment and install dependencies:
   - `python -m venv .venv`
   - `source .venv/bin/activate`
   - `pip install -e .[dev]`
3. Run tests: `pytest`

## Guidelines

- Keep changes focused and small when possible.
- Add or update tests for behavior changes.
- Prefer clear, explicit names and minimal dependencies.
- Document user-facing changes in `README.md` when needed.

## Reporting issues

Please include:

- OS and Python versions
- Command you ran and full error output
- Steps to reproduce

## Code of Conduct

By participating, you agree to abide by the Code of Conduct.
